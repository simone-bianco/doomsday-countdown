# Glossary

**Catalog chapter** — chapter that maps existing package exports by component family.

**Package component** — reusable component in `packages/simone-bianco/vue-ui-components`.

**Testing package** — `packages/simone-bianco/vue-ui-components-testing`, used for demo/manual validation pages.

**UI prop** — typed class override object merged with component defaults.

**`cn()`** — class merge utility used to combine base classes and overrides.
