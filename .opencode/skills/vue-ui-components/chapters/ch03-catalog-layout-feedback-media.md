# Catalog: Layout, Feedback, Media & Navigation

## Load When
Choosing or reviewing layout shells, status/feedback UI, navigation, modals, cards, loading, and organisms.

## Layout and content

- `Card` — card container.
- `PageHeader` — page heading/actions.
- `SectionPanel` — section/card panel.
- `StatCard` — metric/status card.
- `EmptyState` — empty collection/status display.
- `CollapsibleSection` — expandable section.
- `ExpandableContent` — collapsed/expanded content.
- `CodeBlock` — formatted code display.

## Feedback and state

- `Modal` — base modal.
- `WarningModal`, `DangerModal`, `InfoModal` — semantic modal variants.
- `ToastNotification` — global toast display.
- `StatusBadge` — status label.
- `StatusDot` — compact status indicator.
- `Loader` — loading indicator.
- `SkeletonLoader` — skeleton placeholder.
- `UsageBar`, `UsageBars` — usage/progress bars.
- `EventLog`, `EventLogItem` — event timeline/log.

## Navigation and page transitions

- `NavLink` — nav link.
- `ResponsiveNavLink` — responsive nav link.
- `Dropdown`, `DropdownItem`, `DropdownLink` — dropdown family.
- `Tabs`, `TabPanel` — tab system.
- `PageTransition` — page transition wrapper.
- `SearchableDropdown` — searchable dropdown.

## Organisms/theme

- `SidebarNavigation` — sidebar navigation organism.
- `Topbar` — topbar organism.
- `ThemeProvider` — theme context/provider.
