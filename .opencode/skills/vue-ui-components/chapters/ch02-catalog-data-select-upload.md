# Catalog: Data, Select & Upload

## Load When
Choosing or reviewing tables, search, pagination, select inputs, file upload, image upload, and avatar upload.

## Data and tables

- `DataTable` — client/local table.
- `ServerDataTable` — server-driven table.
- `DataTableRowActions` — row action menu/actions.
- `Pagination` — pagination controls.
- `SearchBox` — search input helper.

## Select family

- `Select` — single select.
- `SelectMultiple` — multi select.
- `SelectChip` — selected option chip rendering.

## Upload family

- `FileUploader` — full file upload component.
- `FilePreview` — uploaded/selected file preview.
- `FileDropZone` — drop-zone surface.
- `ImageUploader` — image-specific upload.
- `AvatarUploader` — avatar-specific upload.
- `AvatarInitials` — avatar placeholder from initials.

## Media data helpers

- `Image` — image display wrapper.
- `ImageGallery` — image gallery.

## Related composables/types

`useFileUpload`, `formatFileSize`, `validateFile`, `validateFiles`, `generateFileId`, `getFileType`, `isPreviewable`, `createPreviewUrl`, `revokePreviewUrl`, `DataTableColumn`, `DataTableSort`, `SelectOption`, `FileUploaderUI`, `ImageUploaderUI`, `AvatarUploaderUI`, `PaginationUI`.
