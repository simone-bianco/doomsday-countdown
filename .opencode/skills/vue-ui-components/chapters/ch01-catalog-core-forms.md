# Catalog: Core & Form Controls

## Load When
Choosing or reviewing core controls, basic inputs, labels, toggles, chips, badges, and field helpers.

## Core/root controls

- `ApplicationLogo` — app/package logo component.
- `Button` — primary button primitive; prefer over native `<button>`.
- `Icon` — package icon resolver wrapper.
- `Tooltip` — hover/focus contextual help.
- `Badge` — semantic compact label.
- `Chip` — compact selected/item label.

## Form/input controls

- `TextInput` — text input primitive.
- `Textarea` — multiline input.
- `NumberInput` — numeric input.
- `Checkbox` — checkbox control.
- `Toggle` — boolean switch.
- `RadioGroup` — radio options group.
- `InplaceInput` — inline editable text/input.
- `DynamicFieldList` — repeatable field list.
- `CopyUrlField` — read/copy URL field.

## Labels and errors

- `InputLabel` — label component.
- `InputError` — single error display.
- `InputErrors` — multi-error display.

## Related types

`ButtonUI`, `BadgeUI`, `ChipUI`, `ToggleUI`, `RadioGroupUI`, `InplaceInputUI`, `NumberInputUI`, `TextareaUI`, `DynamicFieldListUI`, `CopyUrlFieldUI`, `InputErrorsUI`.
